# Motus

**Motus** is an interactive physics simulation app that lets you record and analyze 1D motion in real-time.

## Features

- **Interactive Motion Control**: Drag a ball along a track to create custom motion patterns
- **Real-Time Recording**: Hold 'R' or the record button to capture position vs. time data
- **Position & Velocity Plots**: Visualize both position and velocity graphs simultaneously
- **Playback & Analysis**: Review recordings with synchronized playback across all views
- **Smoothing Filter**: Apply adjustable smoothing to reduce noise in recorded data
- **Motion Trace**: See the path of motion with a visual trace overlay
- **Responsive Design**: Works seamlessly on desktop and mobile devices
